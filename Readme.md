# FASL MicroMundo Landing

Reimplementacion del frontend de `micromundo.team` como sitio estatico en `React + TypeScript + Vite`, sin dependencia de WordPress, plugins ni panel de administracion.

## Objetivo

Mantener la esencia visual y editorial del sitio original:

- identidad de marca
- estructura multipagina
- buscador local
- animaciones livianas
- catalogo de cursos y equipo
- paginas de contacto, recursos y privacidad

## Stack

- React 19
- TypeScript
- Vite
- React Router

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El contenido vive en `src/data/siteData.ts` y los assets estaticos en `public/media`.
